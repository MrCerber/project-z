## What is Project Z?

Project Z is an **open-source communication platform** built on an **open protocol** ‚ÄĒ designed to be everything Discord promised but never fully delivered.

We are not trying to be *"Discord but faster."*

We are building a platform that **stands for something**.

> *The right to communicate freely, privately, and without fear ‚ÄĒ regardless of which government you live under.*

---

## Why?

Discord has become bloated, privacy-invasive, and increasingly bows to political and governmental pressure.

Existing alternatives (Matrix, Revolt, Element) either sacrifice UX for openness, or openness for UX. **None of them get both right.**

Project Z is built on a simple belief: you shouldn't have to choose between freedom and a great experience.

## Key Features

<details>
<summary><b>ūüĖ•ÔłŹ Three display modes</b></summary>
<br/>

- **Full Mode** ‚ÄĒ standard full-window experience
- **Sidebar Mode** ‚ÄĒ slides out from the side, stays accessible
- **Mini Mode** ‚ÄĒ inspired by Winamp/AIMP, always on screen while you game, work, or browse

No more dragging Discord between screens.

</details>

<details>
<summary><b>ūüĒí Real end-to-end encryption</b></summary>
<br/>

- DMs and private channels ‚Üí full E2E encryption (RSA-2048 + AES-256)
- Public channels ‚Üí transport encryption (TLS 1.3) with full audit trail
- Voice/video ‚Üí SRTP, server-blind even on SFU

</details>

<details>
<summary><b>ūüĆć Self Hosted with Data Residency</b></summary>
<br/>

Run your own server in your own country. Comply with local data laws without depending on us.

- Works identically to Cloud servers ‚ÄĒ same features, same client
- Registered in global Registry so other users can find it
- You own your data, you are responsible for it

</details>

<details>
<summary><b>ūüĒď Open Protocol</b></summary>
<br/>

Fully documented. Anyone can build a compatible client or server.

The official client will always be the best ‚ÄĒ like Chrome for HTTP.

Open protocol = practically impossible to block. No single point of failure.

</details>

<details>
<summary><b>ūüďč Personal Audit Log</b></summary>
<br/>

Every system action related to your account is logged and visible to you. Always. Including actions by the platform itself.

</details>

<details>
<summary><b>ūü§Ė Bot transparency</b></summary>
<br/>

Every bot must display its developer, purpose, and exact permissions. Bots cannot impersonate humans ‚ÄĒ ever.

All bot actions are logged in your personal audit log.

</details>

<details>
<summary><b>ūüé® MySpace-inspired personalization</b></summary>
<br/>

Profiles and server pages can look unique through CSS Sandboxed ‚ÄĒ without threatening client stability.

</details>
