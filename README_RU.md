## Что такое Project Z?

Project Z — **коммуникационная платформа с открытым исходным кодом**, построенная на **открытом протоколе**. Всё, чем Discord обещал стать, но так и не стал.

Мы не пытаемся стать *"Discord, только быстрее."*

Мы строим платформу, которая **за что-то стоит**.

> *Право общаться свободно, приватно и без страха — независимо от того, в какой стране ты живёшь.*

---

## Почему?

Discord стал громоздким, нарушает приватность и всё чаще уступает политическому и правительственному давлению.

Существующие альтернативы (Matrix, Revolt, Element) жертвуют либо удобством ради открытости, либо открытостью ради удобства. **Никому не удаётся совместить и то, и другое.**

Project Z основан на простой идее: не нужно выбирать между свободой и хорошим продуктом.

## Основные ценности

| Ценность | Что это значит |
|---|---|
| 🗣️ **Настоящая свобода слова** | Никакой политической цензуры. Красные линии на основе универсальной человеческой этики, а не политических трендов |
| 🔍 **Структурная прозрачность** | Политический контент, платное продвижение и боты всегда видны с пометкой — никакого скрытого влияния |
| 🔐 **Твои данные принадлежат тебе** | Право на полное удаление. Личный аудит-лог — ты всегда знаешь, что система сделала с твоими данными |
| 🛡️ **Без капитуляции** | Мы сотрудничаем с правоохранителями по реальным преступлениям. Мы не уступаем политическому давлению ни от одного правительства или организации |
| 📱 **Один клиент для всего** | Не нужно скачивать новое приложение для каждого сообщества. Один клиент, один опыт |

## Ключевые возможности

<details>
<summary><b>🖥️ Три режима отображения</b></summary>
<br/>

- **Full Mode** — стандартное полноэкранное окно
- **Sidebar Mode** — выдвигается сбоку, всегда доступен
- **Mini Mode** — вдохновлён Winamp/AIMP, всегда на экране пока ты играешь, работаешь или сёрфишь

Никаких перетаскиваний Discord между мониторами.

</details>

<details>
<summary><b>🔒 Настоящее сквозное шифрование</b></summary>
<br/>

- DM и приватные каналы → полное E2E шифрование (RSA-2048 + AES-256)
- Публичные каналы → транспортное шифрование (TLS 1.3) с полным журналом аудита
- Голос/видео → SRTP, сервер не читает даже на SFU

</details>

<details>
<summary><b>🌍 Self Hosted с поддержкой Data Residency</b></summary>
<br/>

Запусти свой сервер в своей стране. Соответствуй местным законам о данных, не завися от нас.

- Работает идентично Cloud серверам — те же функции, тот же клиент
- Регистрируется в глобальном Registry — другие пользователи могут найти
- Твои данные — твоя ответственность

</details>

<details>
<summary><b>🔓 Открытый протокол</b></summary>
<br/>

Полностью задокументирован. Любой может создать совместимый клиент или сервер.

Официальный клиент всегда будет лучшим — как Chrome для HTTP.

Открытый протокол = практически невозможно заблокировать. Нет единой точки отказа.

</details>

<details>
<summary><b>📋 Личный аудит-лог</b></summary>
<br/>

Каждое системное действие, связанное с твоим аккаунтом, записывается и видно тебе. Всегда. Включая действия самой платформы.

</details>

<details>
<summary><b>🤖 Прозрачность ботов</b></summary>
<br/>

Каждый бот обязан показывать разработчика, назначение и точные права. Боты не могут притворяться людьми — никогда.

Все действия бота записываются в твой личный аудит-лог.

</details>

<details>
<summary><b>🎨 Персонализация в духе MySpace</b></summary>
<br/>

Профили и страницы серверов могут выглядеть уникально через CSS Sandboxed — без угрозы стабильности клиента.

</details>

## Политика модерации

Мы не полиция. Но мы ответственные граждане.

```
Красные линии (абсолютные):
  ✗  Контент с несовершеннолетними (до 18 лет)
  ✗  Планирование терроризма
  ✗  Торговля людьми

При нарушении:
  → Снапшот доказательств (зашифрован)
  → Жалоба в соответствующие органы (Интерпол, ФБР, Европол, местная полиция)
  → Передача доказательств — дальше их ответственность

Принцип симметрии (применяется ко всем):
  ✓  Преступники                      → удаление
  ✓  Правительство без ордера         → отказ
  ✓  Политическое давление            → отказ
  ✓  Корпоративное давление           → отказ
```
